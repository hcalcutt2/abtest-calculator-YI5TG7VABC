# A/B Test Calculator

A frequentist A/B/n test calculator built with React and Vite.

## Features
- Pre-test planning (sample size and duration)
- Post-test analysis (conversion rate and revenue)
- Holm–Bonferroni correction for multiple variants
- Chi-square SRM check
- Export results to CSV and PDF

## Deployment
This project is automatically deployed to GitHub Pages via GitHub Actions.
